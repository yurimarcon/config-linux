### Como jagar Tron.
Basta todar no terminal o seguinte comando:

```sh
ssh sshtron.zachlatta.com
```

Esse é um jogo online que se joga com as teclas WASD, se você
utilizar os direcionais o jogo ecerra.
Sua cor estará no topo da janela.
